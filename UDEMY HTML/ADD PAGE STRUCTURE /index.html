A programmer is a person who writes, tests, and modifies code to create instructions for computers to follow. Programmers use specialized languages to communicate with computers and other systems to get them to perform specific tasks. Programmers can work in a variety of areas, including: Developing mobile applications, Coding video games, Programming websites, Updating financial records, and Duplicating conditions for pilots training in a flight simulator. Programmers are detail-oriented and must be able to focus on code for long periods of time. They also need to be able to persist to solve code issues that can have a big impact on the program.
